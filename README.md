BOJ_Solution(공사중)
===========

- BOJ(BAEKJOON Online Judge) solution by timetobye
- BOJ ID : time_to_bye
- language : Python3
- 부족한 풀이지만 누군가에게 도움이 되길 바라며.
  - 도움의 손길이 필요한 파이썬 초보입니다.
  - 풀이 내역을 보시고 거침 없는 의견 부탁드려요. 데헷~!
